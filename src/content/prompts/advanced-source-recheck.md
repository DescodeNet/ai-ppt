추가 수집된 소스들을 다시 점검해줘.

검토 목적:
1. 각 소스가 2025년 이후의 최신 정보인지 확인
2. 각 소스에 구체적인 수치, 시장 통계, 공식 데이터가 포함되어 있는지 확인
3. 최신성이 부족하거나 정성적 해석만 있는 소스를 구분

판정 기준:
- 발행일이 2025년 1월 1일 이후인지 확인
- 본문에서 인용한 데이터/통계의 기준 시점이 2025년 이후인지 확인
- 단순 의견/해석이 아니라, 구체적인 수치(예: %, 금액, 성장률, 시장규모, 가입률, 해지율, 손해율, 금리, CPI, 환율 등)가 있는지 확인
- 시장 통계가 있다면 출처가 공식기관, 협회, 연구기관, 통계기관, 감독기관 등 신뢰 가능한 곳인지 확인
- 발행일은 최신이지만, 데이터 기준 시점이 오래된 경우 별도로 표시

표 형식으로 정리:
- 소스명
- 출처
- 발행일
- 2025년 이후 발행 여부 (Y/N)
- 데이터 기준 시점
- 데이터 기준도 2025년 이후인지 (Y/N/불명)
- 구체적 수치 포함 여부 (Y/N)
- 시장 통계 포함 여부 (Y/N)
- 사용된 주요 수치 또는 통계 예시
- 통계/수치 출처 명확성 (높음/보통/낮음)
- 최신성 종합 판정 (최신 / 부분 최신 / 구형 / 불명)
- 활용 우선순위 (상/중/하)

추가로 마지막에 아래도 정리해줘:
1. 최신성이 높은 핵심 소스 TOP 10
2. 발행일은 최신이지만 데이터가 오래된 소스
3. 수치/통계가 부족해서 근거력이 약한 소스
4. 추가 보강이 필요한 통계 주제 5개
