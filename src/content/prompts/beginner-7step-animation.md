"고객 신뢰를 높이는 보험 상담 기법" 발표의 메시지 전달력을 높일 수 있는 애니메이션과 전환 효과 아이디어를 슬라이드별로 추천해줘. 효과 사용법도 포함해줘.

발표 환경: Canva 발표 모드 또는 PowerPoint
슬라이드 수: 10~12장

아래 형식으로 각 슬라이드마다 추천해줘:

- 슬라이드 번호 / 제목
- 추천 전환 효과 (슬라이드 간 이동): 이름 + 이유
- 추천 애니메이션 (슬라이드 내 요소 등장): 이름 + 적용 대상 + 이유
- 타이밍 (클릭 시 / 자동 / 딜레이)

주의사항:
- 과하지 않게 — 효과는 메시지를 돕는 수단
- 데이터 슬라이드: 숫자가 순차적으로 나타나는 효과 권장
- 롤플레이/실습 슬라이드: 주의를 환기하는 효과
- 마무리: 차분한 페이드
