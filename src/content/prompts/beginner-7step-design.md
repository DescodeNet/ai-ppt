모던하고 세련된 디자인으로 "고객 신뢰를 높이는 보험 상담 기법" 발표에 어울리는 PPT 디자인 가이드를 만들어줘. 색상, 폰트, 레이아웃까지 구체적으로 설명해줘.

분위기: 신뢰감 있고 전문적이면서도 따뜻한 느낌
청중: 보험영업팀 설계사

아래 항목을 모두 포함해줘:

1. 색상 팔레트
   - 주 색상 (배경/제목): HEX 코드 포함
   - 보조 색상 (본문/소제목): HEX 코드 포함
   - 포인트 색상 (강조/CTA): HEX 코드 포함
   - 색상 사용 규칙 (어디에 어떤 색을 쓰는지)

2. 폰트
   - 제목 폰트: 구체적 이름과 크기
   - 본문 폰트: 구체적 이름과 크기
   - 강조 텍스트 처리법 (굵기, 색상, 밑줄 중 택1)

3. 레이아웃
   - 제목 슬라이드 레이아웃
   - 내용 슬라이드 레이아웃 (텍스트 + 이미지 배치)
   - 데이터 슬라이드 레이아웃 (차트 위치)
   - 여백·정렬 규칙

4. 시각 요소
   - 아이콘 스타일 (라인/솔리드, 추천 소스)
   - 이미지 톤 (밝은/어두운, 필터)
   - 구분선·도형 활용법
