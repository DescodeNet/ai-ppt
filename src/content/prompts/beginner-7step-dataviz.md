보험영업팀 교육 발표에서 사용할 아래 데이터를 효과적으로 시각화할 수 있는 그래프나 차트 유형을 추천해줘. 데이터 분석 결과를 명확하게 전달하는 방법도 설명해줘.

시각화할 데이터:
1. 초회 상담 → 계약 전환율 변화 (월별, 최근 6개월)
2. 고객 이탈 사유 비율 (가격 / 서비스 불만 / 타사 제안 / 기타)
3. 신뢰 형성 기법 적용 전/후 고객 만족도 비교
4. 설계사별 소개 고객 비율 상위 5명
5. 상담 단계별 소요 시간 (첫인사 → 니즈파악 → 보장분석 → 클로징)

각 데이터에 대해:
- 추천 차트 유형 (막대/꺾은선/도넛/퍼널 등)
- 선택 이유
- 색상 활용 팁 (강조할 부분에 포인트 색상)
- 라벨·범례 배치 방법
- Canva에서 만드는 방법 (있다면)

조건:
- 숫자가 없으면 가상의 현실적인 예시 데이터를 넣어줘
- 한눈에 핵심이 보이도록 단순하게
