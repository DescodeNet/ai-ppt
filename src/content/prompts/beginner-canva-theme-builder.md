아래 조건에 맞는 **발표용 테마(템플릿)**를 먼저 만들고, 그 테마를 적용한 슬라이드를 생성해줘.

## 1단계: 테마 설정

브랜드/발표 정보:
- 발표 주제: <<<예: AI 도구 활용 교육>>>
- 분위기: <<<예: 전문적이고 깔끔한 / 따뜻하고 친근한 / 모던하고 미니멀한>>>
- 주 색상: <<<예: 남색(#1B3A5C) + 흰색 / 자동 추천>>>
- 보조 색상: <<<예: 포인트 오렌지(#FF6B35) / 자동 추천>>>
- 폰트 스타일: <<<예: 고딕 계열 깔끔한 폰트 / 자동 추천>>>

테마 규칙:
- 모든 슬라이드에 동일한 색상 팔레트, 폰트, 레이아웃 규칙 적용
- 제목 슬라이드, 내용 슬라이드, 구분 슬라이드, 마무리 슬라이드 4가지 레이아웃 포함
- 여백과 정렬이 일관되게 유지
- 텍스트 계층: 제목(굵고 크게) → 소제목(중간) → 본문(가독성 우선)
- 포인트 색상은 강조할 키워드나 수치에만 사용

## 2단계: 슬라이드 생성

위 테마를 적용하여 아래 발표를 만들어줘:
- 슬라이드 수: <<<8~12장>>>
- 청중: <<<예: 직장인 / 대학생 / 경영진>>>
- 목적: <<<예: AI 도구 소개 / 분기 보고 / 교육>>>

슬라이드 구성:
1. 표지 — 발표 제목 + 발표자 이름 + 날짜
2. 목차 — 전체 흐름 요약
3~N. 본문 — 슬라이드당 핵심 메시지 1개, 보조 설명 2~3줄 이내
- 마무리 — 요약 또는 Q&A

조건:
- 한국어로 작성
- 슬라이드마다 불필요한 장식 없이 핵심만
- 이미지는 주제에 맞는 것으로 자동 배치
- 차트/수치가 필요한 슬라이드는 데이터 시각화 레이아웃 사용

## 추천 활용 팁
- 완성 후 '브랜드 키트'에 색상·폰트를 저장하면 다음 발표에도 동일 테마 재사용 가능
- 이미지 교체: '요소' 탭에서 드래그&드롭
- AI 이미지: '앱' → 'Magic Media'로 맞춤 이미지 생성
- 배경 제거: '사진 편집' → 'BG Remover'로 누끼 따기
- PPTX 내보내기: '공유' → '다운로드' → 'Microsoft PowerPoint' 선택
