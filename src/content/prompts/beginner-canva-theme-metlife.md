아래 조건에 맞는 발표용 테마(템플릿)를 먼저 만들고, 그 테마를 적용한 슬라이드를 생성해줘.

1단계: 테마 설정

브랜드/발표 정보:
- 발표 주제: AI 도구를 활용한 업무 효율화 — 팀 내 교육용
- 분위기: 전문적이고 깔끔한
- 주 색상: 메트라이프 블루(#00539B) + 흰색(#FFFFFF)
- 보조 색상: 메트라이프 그린(#00A94F)
- 폰트 스타일: 고딕 계열 깔끔한 폰트

테마 규칙:
- 모든 슬라이드에 동일한 색상 팔레트, 폰트, 레이아웃 규칙 적용
- 제목 슬라이드, 내용 슬라이드, 구분 슬라이드, 마무리 슬라이드 4가지 레이아웃 포함
- 여백과 정렬이 일관되게 유지
- 텍스트 계층: 제목(굵고 크게) → 소제목(중간) → 본문(가독성 우선)
- 포인트 색상은 강조할 키워드나 수치에만 사용

2단계: 슬라이드 생성

위 테마를 적용하여 아래 발표를 만들어줘:
- 슬라이드 수: 10장
- 청중: 팀 내 직원 (보험 설계사 및 사무직)
- 목적: AI 도구 활용법 교육 — 실무에서 바로 쓸 수 있는 도구 소개

슬라이드 구성:
1. 표지 — "AI로 일 잘하기" + 발표자: 홍길동 + 2026.03.13
2. 목차 — 오늘 배울 내용 (4가지 도구 + 실습 + 주의사항)
3. 왜 AI 도구인가? — 반복 업무 시간 40% 절약 수치, Before/After 비교
4. 도구 1: ChatGPT — 이메일·보고서 초안 작성, 고객 응대 스크립트 자동 생성
5. 도구 2: Canva — 발표 자료·홍보물·팀 뉴스레터 5분 만에 제작
6. 도구 3: Gamma/Napkin — PPT 초안 자동 생성, 복잡한 내용 시각화
7. 도구 4: NotebookLM — 사내 매뉴얼·약관 분석, 요약, Q&A 챗봇
8. 실습 안내 — 지금 바로 해보기 (QR코드로 Canva 접속)
9. 주의사항 — 고객 개인정보 입력 절대 금지, 사내 보안 정책 준수, 결과물 검수 필수
10. Q&A — 질문 및 마무리

조건:
- 한국어로 작성
- 슬라이드마다 핵심 메시지 1개, 보조 설명 2~3줄 이내
- 불필요한 장식 없이 깔끔하게
- 이미지는 주제에 맞는 것으로 자동 배치
- 수치/비교가 있는 슬라이드는 데이터 시각화 레이아웃 사용
