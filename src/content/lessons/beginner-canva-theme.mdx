summary: "Canva에서 브랜드 테마(색상·폰트·레이아웃)를 먼저 설정하고, 그 테마를 적용해 슬라이드를 생성하는 실습입니다. 한 번 만든 테마를 브랜드 키트에 저장하면 매번 새로 디자인할 필요가 없습니다."
level: beginner

tools:

prerequisites:
  - Canva(canva.com) 회원가입 및 로그인
  - 크롬 또는 엣지 최신 브라우저
  - "발표 주제 한 줄 준비(예: 팀 내 AI 도구 활용 교육)"
steps:
  - title: "왜 테마를 먼저 만들어야 하는가?"
    body: |
      발표 자료를 만들 때 가장 흔한 실수: 슬라이드마다 색상, 폰트, 레이아웃이 다름 → **비전문적인 인상**.

      테마를 먼저 정하면:
      - 모든 슬라이드가 **동일한 룩앤필**로 통일
      - 새 슬라이드를 추가해도 자동으로 테마 적용
      - 다음 발표에서도 **같은 테마 재사용** 가능 (브랜드 키트)

      기업 교육, 팀 보고서, 프로젝트 제안 등 **반복적으로 발표하는 환경**에 특히 유용합니다.
  - title: "테마 설정 — 색상·폰트·레이아웃 규칙 정하기"
    body: |
      프롬프트를 입력하기 전에 아래 3가지를 정합니다:

      **1. 색상 팔레트 (3색 원칙)**
      - 주 색상: 배경이나 제목에 사용 (예: 남색 #00539B)
      - 보조 색상: 본문·소제목 (예: 흰색/밝은 회색)
      - 포인트 색상: 강조 키워드·수치에만 (예: 초록 #00A94F)

      회사 로고 색상이 있다면 그것을 주 색상으로 쓰는 게 가장 쉽습니다.

      **2. 폰트**
      - 제목: 굵고 큰 고딕 (Canva 추천: Pretendard, Noto Sans KR)
      - 본문: 가독성 우선 (같은 폰트의 Regular 두께)

      **3. 레이아웃 4종**
      - 표지 슬라이드: 제목 + 발표자 + 날짜
      - 내용 슬라이드: 핵심 메시지 1개 + 보조 설명
      - 구분 슬라이드: 섹션 제목만 (전환용)
      - 마무리 슬라이드: 요약 또는 Q&A
  - title: "프롬프트로 테마 적용 슬라이드 생성"
    body: |
      아래 프롬프트 섹션의 **'Canva 일관성 테마 제작 & 슬라이드 생성 프롬프트'**를 복사합니다.

      **사용 방법:**
      1. <<<변수>>> 부분을 본인 주제로 교체
      2. canva.com 홈 검색창에 붙여넣기 → **'Canva AI'** 클릭
      3. 디자인 유형에서 **'프레젠테이션'** 선택
      4. 생성된 결과 중 테마가 일관된 것을 선택
      5. 편집기에서 세부 내용 수정

      **핵심**: 프롬프트에 색상·폰트·레이아웃 규칙을 명시했기 때문에 Canva AI가 일관된 테마로 생성합니다.
  - title: "추천 도구 — 이미지 변환·편집"
    body: |
      슬라이드에 들어갈 이미지를 손쉽게 준비하는 도구들:

      **Canva 내장 기능:**
      - **Magic Media**: 텍스트로 이미지 생성 (좌측 '앱' → 'Magic Media')
      - **BG Remover**: 사진 배경 제거/누끼 따기 ('사진 편집' → 'BG Remover')
      - **Magic Eraser**: 사진 속 불필요한 요소 지우기
      - **Magic Resize**: 슬라이드 크기를 SNS, 포스터 등으로 즉시 변환

      **외부 추천 도구:**
      - **remove.bg**: 무료 배경 제거 (remove.bg)
      - **Unsplash/Pexels**: 무료 고화질 이미지 (Canva 내 '사진' 탭에서도 접근 가능)
      - **Flaticon**: 무료 아이콘 (flaticon.com)
      - **Carbon**: 코드 스니펫을 예쁜 이미지로 변환 (carbon.now.sh)
  - title: "브랜드 키트에 저장 → 재사용"
    body: |
      만든 테마를 저장해두면 다음 발표에서도 동일한 디자인을 바로 적용할 수 있습니다.

      1. Canva 홈 좌측 **'브랜드'** 메뉴 클릭
      2. **'브랜드 키트'** → 색상·폰트·로고 등록
      3. 다음 프레젠테이션 생성 시 → 편집기 좌측 **'스타일'** 탭 → 저장된 브랜드 키트 클릭 → 전체 적용

      무료 계정: 브랜드 키트 1개 저장 가능. Pro 계정: 여러 개 저장 가능.
  - title: "PPTX로 내보내기"
    body: |
      1. 우측 상단 **'공유'** → **'다운로드'**
      2. 파일 형식에서 **'Microsoft PowerPoint(.pptx)'** 선택
      3. 다운로드 클릭

      PowerPoint에서 열어 추가 편집하거나 팀원에게 공유할 수 있습니다.
