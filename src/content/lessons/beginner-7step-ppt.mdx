---
title: "7단계 AI 프롬프트로 발표 자료 완성하기"
summary: "콘텐츠 생성부터 Q&A 준비까지, 7개의 프롬프트를 순서대로 사용해 완성도 높은 발표 자료를 만드는 초급 실습입니다. 보험영업팀 실전 예시로 진행합니다."
level: beginner
order: 4
tools:
  - ChatGPT
  - Canva
  - Gamma
prerequisites:
  - ChatGPT(chatgpt.com) 또는 Gemini(gemini.google.com) 접속
  - Canva(canva.com) 또는 Gamma(gamma.app) 회원가입
  - 크롬 또는 엣지 최신 브라우저
steps:
  - title: "전체 흐름 이해하기 — 7단계 프롬프트 체인"
    body: |
      발표 자료를 처음부터 끝까지 AI로 만들 때, 한 번에 완벽한 결과를 기대하면 안 됩니다.
      **7개의 프롬프트를 순서대로** 사용하면 단계별로 품질을 쌓아올릴 수 있습니다.

      | 단계 | 목적 | 사용 도구 |
      |------|------|-----------|
      | 1️⃣ 콘텐츠 생성 | 핵심 내용 초안 | ChatGPT/Gemini |
      | 2️⃣ 슬라이드 구조 | 목차와 슬라이드별 포인트 | ChatGPT/Gemini |
      | 3️⃣ 디자인 가이드 | 색상·폰트·레이아웃 결정 | ChatGPT/Gemini |
      | 4️⃣ 애니메이션·전환 | 슬라이드별 효과 추천 | ChatGPT/Gemini |
      | 5️⃣ 데이터 시각화 | 차트·그래프 유형 선택 | ChatGPT/Gemini |
      | 6️⃣ 발표 스크립트 | 슬라이드별 대본 + 스토리텔링 | ChatGPT/Gemini |
      | 7️⃣ Q&A 준비 | 예상 질문·답변·피드백 | ChatGPT/Gemini |

      1~2단계 결과를 Canva나 Gamma에 넣어 슬라이드를 만들고, 3~7단계로 완성도를 높입니다.
  - title: "1단계: 콘텐츠 생성 — 발표 초안 뽑기"
    body: |
      가장 먼저 **무슨 이야기를 할 것인지** 핵심 내용을 뽑아냅니다.

      아래 프롬프트 섹션의 **'① 콘텐츠 생성'** 프롬프트를 ChatGPT에 붙여넣으세요.

      **팁:**
      - 주제가 막연하면 "보험영업팀이 고객 신뢰를 높이는 방법"처럼 구체적으로
      - AI가 생성한 초안에서 불필요한 내용은 바로 삭제
      - 이 단계의 결과물이 나머지 6단계의 재료가 됩니다
  - title: "2단계: 슬라이드 구조 설계 — 목차 잡기"
    body: |
      1단계에서 뽑은 핵심 내용을 **슬라이드 단위로 분배**합니다.

      **'② 슬라이드 구조 설계'** 프롬프트를 사용하세요.

      **팁:**
      - 1단계 결과를 함께 붙여넣으면 더 정확한 구조가 나옵니다
      - 슬라이드 수가 너무 많으면 "10장 이내로 줄여줘"라고 추가 요청
      - 여기서 나온 목차가 발표의 뼈대입니다
  - title: "3단계: 디자인 가이드 — 시각 톤 결정"
    body: |
      내용과 구조가 잡혔으면 **어떤 느낌으로 보여줄지** 결정합니다.

      **'③ 디자인 가이드'** 프롬프트를 사용하세요.

      **팁:**
      - AI가 추천한 색상·폰트를 Canva의 브랜드 키트에 저장
      - 회사 CI 색상이 있다면 프롬프트에 명시 (예: "메트라이프 블루 #00539B 기반으로")
      - 이 결과를 Canva에 넣으면 일관된 테마가 적용됩니다
  - title: "4단계: 애니메이션·전환 효과 — 전달력 높이기"
    body: |
      슬라이드가 단조롭지 않도록 **적절한 효과**를 추가합니다.

      **'④ 애니메이션·전환 효과'** 프롬프트를 사용하세요.

      **팁:**
      - 효과는 "있으면 좋고, 없어도 되는" 수준으로 — 과하면 역효과
      - 데이터 슬라이드: 숫자가 하나씩 나타나는 효과 추천
      - Canva 발표 모드에서 효과 적용 가능
  - title: "5단계: 데이터 시각화 — 숫자를 그림으로"
    body: |
      보고서나 실적 데이터가 있다면 **어떤 차트로 보여줄지** AI에게 물어봅니다.

      **'⑤ 데이터 시각화'** 프롬프트를 사용하세요.

      **팁:**
      - 비교 → 막대 그래프, 추이 → 꺾은선, 비율 → 도넛/파이
      - AI가 추천한 차트 유형을 Canva의 '차트' 요소로 직접 생성 가능
      - 숫자가 없어도 "가상의 예시 데이터로 보여줘"라고 요청 가능
  - title: "6단계: 발표 스크립트 — 대본 만들기"
    body: |
      슬라이드는 시각 자료일 뿐, **실제로 말할 내용**이 발표의 핵심입니다.

      **'⑥ 발표 스크립트'** 프롬프트를 사용하세요.

      **팁:**
      - 슬라이드당 1~2분 분량이 적당
      - 스토리텔링 요소 (실제 사례, 에피소드)가 들어가면 청중 집중도 상승
      - Canva 발표자 노트에 스크립트를 붙여넣어 발표 중 참고 가능
  - title: "7단계: Q&A 준비 — 예상 질문 대비"
    body: |
      발표가 끝난 뒤 나올 **예상 질문과 답변**을 미리 준비합니다.

      **'⑦ Q&A 준비'** 프롬프트를 사용하세요.

      **팁:**
      - 까다로운 질문일수록 준비 가치가 높음
      - "이 발표에 대해 비판적인 관점에서 질문해줘"라고 요청하면 더 날카로운 질문이 나옴
      - Q&A 카드를 별도 슬라이드로 만들어두면 발표 중 참고 가능
  - title: "최종 조립 — Canva/Gamma에서 완성"
    body: |
      7단계 결과물을 종합하여 최종 발표 자료를 만듭니다:

      1. **1~2단계 결과** → Canva 또는 Gamma에 붙여넣어 슬라이드 생성
      2. **3단계 디자인 가이드** → 색상·폰트·레이아웃 적용
      3. **4단계 효과** → 애니메이션 추가
      4. **5단계 차트** → 데이터 슬라이드에 시각화 삽입
      5. **6단계 스크립트** → 발표자 노트에 붙여넣기
      6. **7단계 Q&A** → 마지막 슬라이드 또는 별도 문서로 보관
      7. PPTX 다운로드 → 팀원 공유

  - "각 단계의 프롬프트는 이전 단계 결과를 함께 붙여넣으면 더 좋은 결과가 나옵니다."
  - "한 번에 완벽하게 만들려 하지 말고, 단계별로 다듬어 나가세요."

---

발표 자료는 한 번에 만들 수 없습니다. **7개의 프롬프트를 순서대로** 사용하면 콘텐츠 → 구조 → 디자인 → 효과 → 시각화 → 스크립트 → Q&A까지 체계적으로 완성할 수 있습니다.
