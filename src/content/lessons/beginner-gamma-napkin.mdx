  - "발표 주제 한 줄 준비(예: 사내 AI 교육 도입안)"

steps:
  - title: "발표 주제·청중·목적 한 줄로 정하기"
    body: "누구에게, 왜 발표하는지를 한 줄로 확정합니다. 이 한 줄이 Gamma와 Napkin 모두에 들어갈 핵심 입력입니다."
  - title: "Gamma에서 PPT 초안 생성 → PPTX 다운로드"
    body: |
      아래 프롬프트 섹션의 'Gamma PPT 초안 생성 프롬프트'를 복사합니다. <<<변수>>> 부분을 본인 주제로 바꾼 뒤 Gamma 입력창에 붙여넣고 생성합니다. 완성된 슬라이드를 PPTX로 다운로드합니다.

      **이미지 아트 스타일 선택 가이드** — Gamma에서 생성 시 아트 스타일을 고를 수 있습니다:

      | 스타일 | 특징 | 추천 상황 |
      |--------|------|-----------|
      | **사진** | 실사 이미지, 현실적 | 비즈니스 보고서, 제품 소개 |
      | **일러스트** | 손그림 느낌, 친근함 | 교육 자료, 사내 문화 소개 |
      | **요약** | 도형·아이콘 중심, 깔끔 | 데이터 요약, 전략 발표 |
      | **3D** | 입체 렌더링, 시선 집중 | 기술 발표, 신제품 컨셉 |
      | **라인아트** | 선만으로 표현, 미니멀 | 공식 제안서, 격식 있는 자리 |
      | **사용자** | 직접 업로드한 이미지 사용 | 브랜드 가이드라인이 있을 때 |

      처음이라면 **'사진'** 또는 **'일러스트'**가 무난합니다. 청중이 임원이면 **'요약'** 또는 **'라인아트'**를 추천합니다.
  - title: "Napkin.ai에서 시각화 자료 생성 → 다운로드"
    body: "아래 프롬프트 섹션의 'Napkin.ai 시각화 프롬프트'를 복사합니다. <<<변수>>> 부분을 같은 주제로 바꾼 뒤 Napkin.ai 입력창에 붙여넣고 시각화를 생성합니다. 다이어그램·플로우차트 등 시각 자료를 PNG 또는 PPTX로 다운로드합니다."
  - title: "두 결과물 합치기 — 최종 PPT 완성"
    body: "Gamma에서 다운로드한 PPTX를 PowerPoint에서 열고, Napkin.ai에서 만든 시각 자료를 텍스트만 있던 슬라이드에 삽입합니다. Gamma가 만든 구조 위에 Napkin의 다이어그램이 보조 역할을 해서 완성도가 올라갑니다."
