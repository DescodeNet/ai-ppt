steps:
  - title: "Canva vs PPTX — 차이점 이해하기"
    body: |
      **Canva**는 웹 브라우저에서 바로 디자인하는 도구입니다. PowerPoint(PPTX)와 핵심 차이:

      | 항목 | Canva | PowerPoint(PPTX) |
      |------|-------|-------------------|
      | 디자인 | 수천 개 템플릿 즉시 적용 | 직접 디자인하거나 제한된 템플릿 |
      | 이미지 교체 | 드래그&드롭으로 즉시 교체 | 삽입→크기 조정→위치 잡기 |
      | AI 기능 | Magic Design으로 자동 생성 | Copilot(유료) |
      | 협업 | 링크 공유로 실시간 편집 | 파일 주고받기 |
      | 출력 | PPTX, PDF, 영상 등 다양 | PPTX 중심 |

      핵심: Canva는 **디자인이 이미 되어 있는 상태에서 시작**하기 때문에 비전문가도 빠릅니다.
  - title: "템플릿 선택 → AI로 초안 생성"
    body: |
      **방법 A — Canva 홈에서 바로 시작 (추천)**

      1. canva.com 홈 상단의 검색창에 아래 프롬프트를 붙여넣기
      2. 검색창 아래 나타나는 **'Canva AI'** 버튼 클릭
      3. 디자인 유형에서 **'프레젠테이션'** 선택
      4. 생성된 템플릿 중 마음에 드는 것을 골라 편집 시작

      **방법 B — 편집기 안에서 사용**

      1. Canva 홈 → **'프레젠테이션'** → 빈 프레젠테이션 열기
      2. 좌측 사이드패널 → **'디자인'** 탭 클릭
      3. 상단 검색 필드에 아래 프롬프트를 붙여넣기
      4. **'디자인 생성'** 클릭 → 결과 중 선택

      프롬프트에서 <<<변수>>> 부분을 본인 주제로 바꿔서 입력하세요.
  - title: "이미지·텍스트 교체 실습"
    body: |
      Canva에서 이미지와 텍스트를 바꾸는 건 매우 간단합니다:

      **이미지 교체**: 좌측 '요소' 또는 '사진' 탭에서 원하는 이미지를 기존 이미지 위로 드래그하면 자동 교체됩니다.

      **텍스트 수정**: 텍스트를 클릭하고 바로 타이핑. 폰트·크기·색상은 상단 툴바에서 변경.

      **AI 이미지 생성**: 좌측 '앱' → 'Magic Media' → 원하는 이미지를 텍스트로 설명하면 AI가 생성해줍니다.
  - title: "PPTX로 내보내기"
    body: |
      1. 우측 상단 **'공유'** → **'다운로드'**
      2. 파일 형식에서 **'Microsoft PowerPoint(.pptx)'** 선택
      3. 다운로드 클릭

      다운로드한 PPTX 파일을 PowerPoint에서 열어 추가 편집하거나, Gamma/Napkin 결과물과 합칠 수 있습니다.

Canva의 핵심은 **디자인을 고르는 것**입니다. AI가 초안을 만들어주고, 템플릿이 레이아웃을 잡아주니 내용만 채우면 됩니다.
