## 난이도 누수 방지 규칙
- beginner에는 딥리서치 편향 점검(-2~+2), 컴플라이언스 리스크 토론 프롬프트를 넣지 않는다.
- advanced 전용 경고(Callout)를 배치해 초급/중급 기본 경로와 분리한다.
- prompt library 필터 기본값은 beginner 포함 전체이며, advanced는 명시 선택 시 노출이 강화된다.
